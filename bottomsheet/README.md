# bottomsheet

----------------------------------------

### FloatingButton을 클릭하면 하단에서 TextField가 포함된 BottomSheet가 올라옵니다.
-MediaQuery를 이용하여 BottomSheet의 패딩 값을 줘서 TextField를 클릭하면 BottomSheet가 하단에서 올라오는 키보드의 높이만큼 더 올라갑니다.
