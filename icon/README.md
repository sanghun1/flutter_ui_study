### clip
- ClipRRect : 이미지 끝을 동그랗게 자른다.
- ClipOval : 이미지를 동그랗게 자른다.
- ClipRect : 이미지를 네모로 자른다.
- ClipPath : Custom으로 모양을 지정해서 자른다.
